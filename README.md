# Project-Air-Quality-Index-Calculator

Aim:

To predict AQI given the values of 6 pollutants:

1. PM2.5

2. PM10

3. NO2

4. NOx

5. NH3

6. CO

Skills used:

1. Python
2. Machine Learning
3. Random Forest Regressor (Ensembling Technique)
